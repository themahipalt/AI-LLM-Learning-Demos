# 06 - Text as Data

## Purpose

This demo explains how human text becomes machine-readable data before it can be processed by Machine Learning, Deep Learning, or LLM systems.

## Learning Flow

Human Text  
↓  
Clean / Prepare Text  
↓  
Split into Words  
↓  
Create Vocabulary  
↓  
Convert Words to IDs  
↓  
Preserve Sequence  
↓  
Create Simple Vector Representation  

## What this demo teaches

- Text is unstructured data
- Text must be converted into numbers
- Vocabulary stores known text pieces
- Each word/token can be assigned an ID
- Text order must be preserved
- Vectors are used so neural networks can process text

## Important Note

This is a simplified educational demo.

Real LLMs use:

- Tokenization
- Token IDs
- Learned embeddings
- Positional information
- Attention
- Transformer blocks
