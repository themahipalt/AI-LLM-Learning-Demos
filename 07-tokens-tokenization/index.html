3. Word Tokenization

Word tokenization breaks text by spaces and punctuation. This is simple, but real LLM tokenizers are more advanced.

4. Subword-like Tokenization

Modern LLMs mostly use subword tokenization. Common words may stay whole, but larger or complex words may be broken into smaller pieces.

5. Token ID Table

Each token gets a numeric ID. These IDs are lookup numbers, not mathematical meaning.

6. Teaching Summary

Tokenization is the first step in the LLM pipeline. It converts human text into small pieces called tokens. These tokens are mapped to token IDs. Token IDs are then converted into embeddings, which are processed by the neural network.

Remember:
Tokens decide how much the model can read, how much it can write, how fast it responds, and how much the API may cost.
